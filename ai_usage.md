# AI Usage

I used ChatGPT as a coding assistant while building this project.

Main areas where I used it:
- Clarifying the object model for Market, OrderBook, Order, Trade, and Position
- Thinking through price-time priority
- Reasoning about partial fills
- Debugging import issues
- Planning FastAPI endpoints
- Drafting README structure

I wrote and tested the implementation locally, including testing through the Python REPL and FastAPI Swagger docs.
